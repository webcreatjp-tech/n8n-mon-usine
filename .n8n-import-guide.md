# Guide d'import n8n

## Prérequis

- n8n self-hosted (v1.0+)
- Les credentials configurés (voir README)

## Étapes d'import

1. Ouvrir n8n dans votre navigateur
2. Aller dans **Workflows** (menu latéral)
3. Cliquer sur **Import** (bouton en haut à droite)
4. Sélectionner le fichier JSON à importer
5. Vérifier et configurer les credentials manquants
6. Sauvegarder et activer le workflow

## Ordre d'import recommandé

Importer les workflows dans l'ordre numérique (00, 01, 02...) pour respecter les dépendances entre sous-workflows.

## Variables d'environnement

Certains workflows utilisent des variables n8n. Les configurer dans :
**Settings → Variables** avant d'activer les workflows.

## Résolution de problèmes

- **Credential manquant** : créer le credential du bon type, puis le rattacher au node concerné
- **Webhook URL** : après import, l'URL du webhook change — mettre à jour les services externes
- **Sub-workflow introuvable** : vérifier que le workflow parent est importé avant l'enfant
